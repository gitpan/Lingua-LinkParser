 TW 
TW connects days of the week to month names in sentences
like this one:

                	 +---TW---+----Xc---+
		    +-ON-+   +-Xd-+--TM-+   |
		    |    |   |    |     |   |
the meeting.n is.v on Monday , January 21 ///// 

See "DT" for more discussion of time expressions.
